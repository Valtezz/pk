Инструкция по использованию скриптов

Настройка ноутбука делается достаточно легко:

Закинуть скрипт на флешку
1. Включить ноутбук
2. Открыть флешку нажать f4
3. В открывшейся консоли ввести "su-" (без кавычек) нажать энтер ввести пароль "12345678"
4. ввести "sh "(с пробелом) и мышкой перетащить файл скрипта настройки в терминал нажать энтер
5. в открывшемся окне переименовать ноутбук по шаблону n2095-tek-(порядковый номер)
6. ждать пока скрипт все сделает сам и выключит компьютер

Стандартные пароли которые задает скрипт

Для ноутбуков:
root: rjvuytoer
teacher: 12345678
admin: callmeadmin
Для Моноблоков:
root: rjvuytoer
teacher: callmeadmin
student: student
Шаблон для кабинета информатики (из инструкции дита):
sm2095-tek-(№ кабинета)-(порядковый №)
Для моноблока учителя убрать букву s из названия и порядковый номер 0(ноль)
